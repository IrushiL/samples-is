Following is the folder structure of the WSO2 Identity Server Quick Start Guide (IS-QSG).

bin - Contains all required scripts to run QSG scenarios
conf - Configuration file(s) required to run the samples
lib - Contains JARs required for QSG
scenarios - QSG scenario related SOAP messages. These are required for IS setup during the runtime
webapps - Contains web applications used in QSG delivered as .war.
          Note that they can be deployed in minimalistic JAVA EE compatible server (ex - Tomcat)
